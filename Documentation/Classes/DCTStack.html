Overview

This represents a stack (last in, first out) data structure in computer science. It has the push and pop operations usually found in stacks, plus a few other methods to make it more useful.

Tasks

Standard stack operations

  • - count The number of items on the stack.
  • - push: Pushes the given object onto the top of the stack.
  • - pop Removes the top object from the stack.
  • - top Returns the top object without popping it from the stack.

Extra functionality

  • - allObjects Returns an NSArray of all the objects in the stack.
  • - popToObject: Pops the stack until it reaches the given object.
  • - popToRootObject Pops the stack until it has only one item left.
