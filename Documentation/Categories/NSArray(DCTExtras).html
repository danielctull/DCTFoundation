dct_splitArrayIntoArraysWithCount:

Splits the receiver into sub-arrays of the given count.

- (NSArray *)dct_splitArrayIntoArraysWithCount:(NSInteger)aCount

Parameters

aCount
The maximum length each of the subarrays should be.

Return Value

An array of the sub-arrays.

Discussion

Splits the receiver into sub-arrays of the given count.

If the receiver has 12 objects, and you give a count of 5 you will be returned an array with 3 arrays inside, with counts of 5, 5 and 2. If you gave a count of 6, you would be returned an array with 2 sub-arrays, both with a count of 6.
